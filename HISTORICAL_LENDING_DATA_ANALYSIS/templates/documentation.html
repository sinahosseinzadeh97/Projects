User Guide

This documentation provides guidance on using the Loan Prediction System.

Loan Prediction Tool

The main feature of the system is the loan prediction tool, which allows you to input borrower and loan information to receive a prediction on loan performance.

  1. Input Form: Fill out all required fields in the loan prediction form.
  2. Prediction Results: After submitting the form, you'll receive a prediction indicating whether the loan is likely to be in "Good Standing" or present a "Default Risk".
  3. Probability Score: The system provides a probability score indicating the confidence level of the prediction.

Input Fields

  • Loan Amount: The total amount of the loan in dollars.
  • Interest Rate: The annual interest rate as a percentage.
  • Term: The loan duration in months.
  • Loan Grade: The assigned grade of the loan (A through G, with A being the highest quality).
  • Employment Length: The borrower's employment length in years.
  • Annual Income: The borrower's annual income in dollars.
  • Debt-to-Income Ratio: The borrower's debt-to-income ratio as a percentage.
  • Income Verification Status: Whether the borrower's income has been verified.
  • Home Ownership: The borrower's home ownership status (Own, Mortgage, Rent, Other).
  • Total Credit Lines: The total number of credit lines in the borrower's credit history.
  • Open Credit Lines: The number of open credit lines.
  • Number of Mortgage Accounts: The number of mortgage accounts.
  • Paid Principal: The amount of principal paid on existing loans.
  • Paid Total: The total amount paid on existing loans.

Technical Documentation

The prediction model is built using machine learning algorithms trained on historical lending data.

  • Machine learning models include Random Forest, Gradient Boosting, and Logistic Regression
  • Features selected based on importance and correlation with loan outcomes
  • Optimized for balanced precision and recall
  • Regular updates to incorporate new data and improve accuracy
